# FiveM-External Base
その名の通り、FiveMの外部チートのベースです。  
現状、b2944のみサポートされています。

## 仕様
* OpenProcess
* Read/WriteProcessMemory
* ImGui TOMOST Overlay

現状特に問題はありませんが、必要に応じて置き換えてください。
